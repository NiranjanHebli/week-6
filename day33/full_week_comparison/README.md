# Day 33 - PM - SVM and KNN & Full Week Comparison

## Part A - Concept Application

This task created a machine learning cheat sheet notebook covering eight algorithms: Logistic Regression, Decision Tree, Random Forest, Extra Trees, K-Nearest Neighbors, Support Vector Machine, K-Means clustering, and XGBoost. Each algorithm was evaluated on a common dataset using cross-validation, and models were ranked based on performance and practical suitability.

## Key Learnings

- Ensemble models like XGBoost, Random Forest, and Extra Trees usually gave the best accuracy on tabular data.
- Logistic Regression worked well as a simple and interpretable baseline.
- Support Vector Machine and K-Nearest Neighbors required feature scaling for better performance.
- Decision Tree was easy to understand but more prone to overfitting.
- Cross-validation helped compare models fairly and select the most reliable one.
